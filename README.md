Dieses Repository enthält die wichtigsten Programm, Testungen zur Bachelorarbeit.

KI-gestützter Speech-to-Text-Transkription.

- **Programme/** – Quellcode und Skripte (z. B. Transkriptionsworkflow)  
- **Testungen/** – Tests und Auswertungen
